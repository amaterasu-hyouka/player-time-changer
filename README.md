# PlayerTimeChanger

## プラグイン説明
Minecraft 1.21.11 Bukkit/Spigot/Paper系サーバー向けのプラグイン

## 機能

プレイヤーごとに時間を設定できます。
他のプレイヤーの時間には影響せず、それぞれ異なる時間を表示できます。

### 使用方法

時計を右クリック

### slotの対応

0 -> 処理なし

1 -> 23000 朝焼け  
2 -> 0 朝  
3 -> 6000 昼  
4 -> 12000 夕方  
5 -> 18000 夜  

6 -> 処理なし

7 -> ワールドの時間に戻す
